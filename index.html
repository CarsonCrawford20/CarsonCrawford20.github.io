Carson Crawford

Emerging Tech & Multi-Media Generalist

Hello and welcome! My name is Carson Crawford and I was born and raised in a small resort town in central Idaho called McCall. I've been passionate about all things game design, website design, and coding since my computer science class in my sophomore year in high school. Currently, I am working on my Bachelor's degree at Boise State University for the Games, Interactive Media, and Mobile program. Between each academic year, I have been working with the Forest Service and a few other conservation groups as a seasonal employee. However, recently I've wanted to focus more on jobs or internships related to my major. Eventually, I hope to work full time in the Virtual Reality industry making training simulations or games, but for now, I'm looking for any type of experience including front and backend development.

Spotify UI Redesign

Spotify UI Redesign Project

2020


Spotify’s interface can be jarring to first time users and can be quite confusing creating a playlist all while searching for music. It’s overwhelming to the fresh customer. This project was aimed to help fix that issue and help integrate the user into a more friendly environment for their first time use. Using Adobe Photoshop and XD, I created a mockup of my redesign. While working on this project, I realized that UX has a lot more to play in applications than originally thought and looking back, there could have been better ways of redesigning the layout. For example the text should have been bigger, same with the ‘like’ and the ‘play’ button

Twitter Data Visualization

2021


Tasked with creating a interactive data visualization using my own data, I conducted observational research on what related topics appeared while searching ‘conspiracy’ on Twitter. For this project I used HTML, CSS, JavaScript, and D3.js.

After conducting the research and creating the visualizations, I found trends in certain data and phrases from different points in time. In other words you can see how real world events affected inclinations in the data.

Spotify API Database Website

2022

Spotify API Database Website

2022


Based off of my Spotify UX Redesign, I took it a step further and implemented a website along with a database to hold all the users and their information. As a new user is added or signed in for the first time, it prompt them to choose what kinds of genres they like. Once the genres are chosen, the next page will show them songs based off the genres they chose from previously to add to their first playlist.

This project was an interesting learning experience because organizing a database along with creating a responsive website displaying the data with limited experience. Although tedious, I found it enjoyable linking tables together through Node.js and Express.js
